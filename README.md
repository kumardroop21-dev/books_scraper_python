# 🍕 Foodpapa Scraper - Python

A Python web scraper that collects food data from foodpapa.com

## 🔍 What it does
- Scrapes **Top 5 food links** from foodpapa.com
- Extracts **Title** and **Link**
- Saves data in a **CSV file**

## 🛠️ Technologies Used
- Python
- Requests
- BeautifulSoup4

## ⚙️ Installation
pip install requests beautifulsoup4

## ▶️ How to Run
python pro_15.py

## 📦 Output
Creates food_top5.csv file with title and link columns
