# 📚 Books Scraper - Python

A Python web scraper that collects book data from [books.toscrape.com](https://books.toscrape.com)

## 🔍 What it does
- Scrapes **70 books** from the website
- Extracts **Title** and **Price** of each book
- Saves all data in a **JSON file**

## 🛠️ Technologies Used
- Python
- Requests
- BeautifulSoup4

## ⚙️ Installation

pip install requests beautifulsoup4


## ▶️ How to Run

python pro_18.py


## 📦 Output
Creates a `books.json` file with this format:
[
    {
        "title": "A Light in the Attic",
        "price": "£51.77"
    },
    ...
]
